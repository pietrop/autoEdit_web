# autoEDIT Desktop Edition

## APP FILE STRUCTURE

- ``/bin``

- ``/bin/components``: binary files

- ``/lib``: modules

- ``/config``: config files

- ``/models``: db data models

- ``/public``

- ``/public/images``: assets

- ``/public/javascript``: js

- ``/public/stylesheets``: css

- ``/routes``: routing and controllers

- ``/views``

- ``/views/partials``

- ``/views/partials/template``

- ``app.js``

- ``package.json``
